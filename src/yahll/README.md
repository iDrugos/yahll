# Yahll - Your Personal AI Coding Assistant

**Name:** Drugos
**Language:** Python
**Description:** Yahll is an AI coding assistant designed to help with various programming tasks and projects. Whether you need to write a simple script or develop complex applications, Yahll got your back.

## Key Features
- **Tool-based Commands:** Execute shell commands using tools like `bash_execute`, making it easy to automate repetitive tasks.
- **File Management:** Create, read, edit, and list files with ease using functions like `write_file`, `read_file`, and `list_directory`.
- **Web Search:** Perform web searches directly from the AI for quick information or research.
- **Clipboard Access:** Easily copy and paste text using `clipboard_read` and `clipboard_write` functions.
