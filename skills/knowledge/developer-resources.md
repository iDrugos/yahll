# Developer Resources Guide

## Overview
A map to the best free resources for developers: books, APIs, cloud services, OSS tools, and knowledge sources.

## Free Books & Courses
**Repo:** [EbookFoundation/free-programming-books](https://github.com/EbookFoundation/free-programming-books) — 40+ languages

Categories:
- Programming books by language (Python, JS, Go, Rust, Java, C++, etc.)
- Online courses and MOOCs
- Interactive tutorials
- Cheat sheets
- Podcasts and screencasts
- Competitive programming problem sets

Search interface available at the project website.

## Free Cloud & Dev Services
**Repo:** [ripienaar/free-for-dev](https://github.com/ripienaar/free-for-dev) — 1,600+ contributors

Key categories with notable free tiers:
| Category | Examples |
|----------|---------|
| Cloud compute | AWS Free Tier, GCP Always Free, Oracle Cloud |
| Git hosting | GitHub, GitLab |
| CI/CD | GitHub Actions (2000 min/mo), CircleCI |
| Databases | PlanetScale, Supabase, MongoDB Atlas |
| Serverless | Vercel, Netlify, Cloudflare Workers |
| Auth | Auth0 (7,500 MAU), Clerk |
| Email | SendGrid (100/day), Resend |
| Monitoring | Datadog (5 hosts), Sentry (5K errors/mo) |
| Storage | Cloudflare R2 (10GB), Backblaze B2 |

**Rule:** Only perpetual free tiers, no time-limited trials.

## Public APIs
**Repo:** [public-apis/public-apis](https://github.com/public-apis/public-apis)

50+ categories. Each entry documents:
- Auth type (None / apiKey / OAuth)
- HTTPS support (Yes/No)
- CORS support (Yes/No/Unknown)

Popular categories:
- Weather, Maps, Geocoding
- Finance, Crypto, Exchange rates
- ML & AI (inference APIs)
- Government & open data
- Entertainment (movies, music, anime, games)
- Developer tools (GitHub API, IP lookup, domain info)
