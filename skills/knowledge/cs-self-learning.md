# Self-Taught CS Curriculum

## Overview
A complete, free, self-paced computer science education using courses from MIT, Stanford, Harvard, and other top universities. Equivalent to an undergraduate CS degree.

## Curriculum Structure

```
Intro CS (start here)
    ↓
Core CS (main body — take in order)
    ↓
Advanced CS (choose specialization)
    ↓
Final Project (capstone)
```

**Estimated time:** ~2 years at 20 hours/week

## Level 1 — Intro CS
Goal: Confirm this is for you, build programming foundations.

| Course | Platform | Language |
|--------|----------|---------|
| CS50P (Python intro) | edX/Harvard | Python |
| How to Code: Simple Data | edX/UBC | Racket |

## Level 2 — Core CS

### Programming
- Systematic Program Design (UBC)
- Programming Languages A/B/C (Coursera/UW) — ML, Ruby, Racket
- Object-Oriented Design (Coursera)

### Math
- Calculus 1A, 1B, 1C (MIT OCW)
- Mathematics for CS (MIT OCW) — discrete math, proofs
- Linear Algebra (MIT 18.06)
- Probability (Harvard Stat110)

### Systems
- Build a Modern Computer from NAND (Coursera) — Nand2Tetris
- Operating Systems (Berkeley CS162)
- Computer Networking (Stanford CS144)

### Theory
- Divide and Conquer, Sorting, Searching (Coursera/Stanford)
- Graph Search, Shortest Paths (Coursera/Stanford)
- Data Structures (UC San Diego)
- Algorithm Design and Analysis

### Applications
- Databases: Relational DBs + SQL (Stanford)
- Machine Learning (Andrew Ng, Coursera)
- Computer Graphics (UC San Diego)
- Software Engineering (MIT 6.031)

### Security
- Cybersecurity Fundamentals (edX/RIT)

### Ethics
- Ethics, Technology and Engineering (TU Delft)

## Level 3 — Advanced CS (Pick a Specialization)
- **Systems**: Distributed Systems, Compilers, OS design
- **Theory**: Complexity theory, Cryptography
- **Applications**: Advanced ML, Robotics, Computational Biology
- **Security**: Network security, Applied cryptography

## Level 4 — Final Project
Build something substantial that demonstrates competency across multiple areas. Document it publicly on GitHub.

## Tips
- Use the OSSU Discord for community support (thousands of active learners)
- Don't skip prerequisites — the curriculum shows dependencies explicitly
- All courses are free to audit; pay only if you need certificates
